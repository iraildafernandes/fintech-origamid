## Fintech 🌐📈

Projeto que simula a visualização do faturamento de uma fintech nos últimos 90 dias, incluindo informações sobre valores de venda, recebimento e processamento na moeda local.

## 🚀 Tecnologias

- React
- TypeScript
- React Router Dom
- Recharts
